## ✅✨Run Locally✨✅

Clone the project

```bash
  git clone https://github.com/Nathal97/Rice-eat.git
```

Go to the project directory

```bash
  cd Server-flask
```

Install dependencies

```bash
  pip install flask // after python installed
```

Start the server

```bash
  python app.py
```
